# Don't Touch 🛡️

**抜毛症（トリコチロマニア）の習慣改善アシスタント**

[한국어](README_KO.md) | [English](../README.md) | [中文](README_ZH.md) | [Español](README_ES.md) | [Русский](README_RU.md)

## はじめに

Don't Touchは、無意識に髪を触ったり抜いたりする習慣（抜毛症/トリコチロマニア）の改善をサポートするデスクトップアプリです。

ウェブカメラで手の動きをリアルタイムで追跡し、手が顔や頭に近づくと警告音を鳴らして、習慣に気づくお手伝いをします。

## 主な機能

- 🎥 **リアルタイム検出** - ウェブカメラで手と顔の位置をリアルタイム追跡
- 🔔 **即座のアラート** - 手が頭に近づくと音と視覚的な警告
- 📊 **統計とトラッキング** - 日別/週別のタッチ回数の記録と分析
- 📅 **カレンダービュー** - 月別の習慣状況を一目で確認
- 🔥 **ストリーク機能** - 連続タッチフリー日数の追跡でモチベーションアップ
- ⚙️ **カスタマイズ設定** - 感度、アラート音、クールダウン時間などを調整可能
- 🌐 **多言語対応** - 韓国語、英語、日本語、中国語、スペイン語、ロシア語
- 🖥️ **システムトレイ** - バックグラウンドで静かに実行
- 🔄 **自動アップデート確認** - 新しいバージョンがリリースされると通知
- 📷 **プレビュー切替** - カメラプレビューをオフにしてCPU節約（検出は継続）

## ダウンロードとインストール

### Windows

1. [Releases](https://github.com/writingdeveloper/dont-touch/releases)ページから`DontTouch_Setup_x.x.x.exe`をダウンロード
2. インストーラーを実行し、セットアップウィザードに従って進める
3. スタートメニューまたはデスクトップショートカットから起動

**インストーラーの機能：**
- デスクトップとスタートメニューのショートカット
- Windows起動時の自動実行オプション
- コントロールパネルから簡単にアンインストール
- 自動アップデート通知

> ⚠️ Windows SmartScreenの警告が表示される場合があります。「詳細情報」→「実行」をクリックしてください。

### 動作要件

- Windows 10/11（64ビット）
- ウェブカメラ（内蔵または外付け）

## 使い方

### 基本的な使い方

1. **アプリを起動** - スタートメニューまたはデスクトップショートカットから起動
2. **モニタリング開始** - 「開始」ボタンをクリック
3. **普段通りに作業** - アプリがバックグラウンドで監視
4. **アラートを受信** - 手が頭に近づくと通知

### 設定の調整

- **感度** - 低いほど敏感に検出
- **トリガー時間** - アラートまでの待機時間
- **クールダウン** - 連続アラート防止のための待機時間

### システムトレイ

- ウィンドウを閉じてもトレイで実行継続
- トレイアイコンをクリックでウィンドウの表示/非表示
- 右クリックでクイックメニューにアクセス

## 統計の確認

- **日別グラフ** - 時間帯別のタッチパターン分析
- **週別グラフ** - 曜日別のタッチ状況
- **カレンダー** - 月別のタッチ回数を視覚化
- **ストリーク** - 連続タッチフリー日数の確認

## よくある質問

### ウェブカメラが動作しません
- 他のアプリがウェブカメラを使用中か確認してください
- ウェブカメラのドライバーが最新か確認してください
- アプリにカメラの権限があるか確認してください

### 検出が敏感すぎる/鈍すぎる
- 設定で感度を調整してください
- トリガー時間を増減してみてください

### アラート音を変更したい
- 設定でアラート音を選択するかミュートできます

## プライバシー

- すべてのデータは**ローカルにのみ保存**されます
- ウェブカメラの映像は**サーバーに送信されません**
- **インターネット接続不要**（オフラインで動作）
