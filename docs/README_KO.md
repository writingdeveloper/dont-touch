# Don't Touch 🛡️

## 소개

Don't Touch는 무의식적으로 머리카락을 만지거나 뽑는 습관(발모벽/트리코틸로매니아)을 교정하는 데 도움을 주는 데스크톱 앱입니다.

웹캠을 통해 실시간으로 손의 움직임을 감지하고, 손이 얼굴이나 머리 근처에 접근하면 경고음을 울려 습관을 인식하게 도와줍니다.

## 주요 기능

- 🎥 **실시간 감지** - 웹캠으로 손과 얼굴 위치를 실시간 추적
- 🔔 **즉각적인 알림** - 손이 머리에 접근하면 소리와 시각적 경고
- 📊 **통계 및 추적** - 일별/주별 터치 횟수 기록 및 분석
- 📅 **캘린더 뷰** - 월별 습관 현황을 한눈에 확인
- 🔥 **스트릭 기능** - 연속 터치-프리 일수 추적으로 동기부여
- ⚙️ **맞춤 설정** - 민감도, 알림 소리, 쿨다운 시간 등 조절 가능
- 🌐 **다국어 지원** - 한국어, 영어, 일본어, 중국어, 스페인어, 러시아어
- 🖥️ **시스템 트레이** - 백그라운드에서 조용히 실행
- 🔄 **자동 업데이트 확인** - 새 버전 출시 시 알림
- 📷 **미리보기 토글** - 카메라 미리보기를 끄고 CPU 절약 (감지는 계속 동작)

## 다운로드 및 설치

### Windows

1. [Releases](https://github.com/writingdeveloper/dont-touch/releases) 페이지에서 `DontTouch_Setup_x.x.x.exe` 다운로드
2. 설치 프로그램을 실행하고 설치 마법사를 따라 진행
3. 시작 메뉴 또는 바탕화면 바로가기로 실행

**설치 프로그램 기능:**
- 바탕화면 및 시작 메뉴 바로가기
- Windows 시작 시 자동 실행 옵션
- 제어판에서 쉬운 제거
- 자동 업데이트 알림

> ⚠️ **Windows 보안 경고**
>
> 이 앱은 아직 코드 서명이 되지 않아 Windows에서 보안 경고가 나타날 수 있습니다:
>
> **Windows SmartScreen:**
> 1. "추가 정보" 클릭
> 2. "실행" 클릭
>
> **Windows 11 Smart App Control:**
> - Smart App Control이 "평가" 또는 "켜기" 모드인 경우 앱이 차단될 수 있습니다
> - 일시적으로 비활성화: 설정 → 개인 정보 및 보안 → Windows 보안 → 앱 및 브라우저 컨트롤 → Smart App Control → 끄기
> - 설치 후 다시 활성화 가능
>
> 🔐 **코드 서명 예정**: 향후 릴리스에서 [SignPath](https://signpath.io)를 통한 오픈소스 코드 서명을 적용하여 이러한 경고를 없앨 예정입니다.

### 요구사항

- Windows 10/11 (64비트)
- 웹캠 (내장 또는 외장)

## 사용 방법

### 기본 사용법

1. **앱 실행** - 시작 메뉴 또는 바탕화면 바로가기에서 실행
2. **모니터링 시작** - "시작" 버튼 클릭
3. **평소처럼 작업** - 앱이 백그라운드에서 감시
4. **경고 받기** - 손이 머리에 접근하면 알림 발생

### 설정 조절

- **민감도** - 낮을수록 더 민감하게 감지
- **트리거 시간** - 경고까지의 대기 시간
- **쿨다운** - 연속 알림 방지를 위한 대기 시간

### 시스템 트레이

- 창을 닫아도 트레이에서 계속 실행
- 트레이 아이콘 클릭으로 창 열기/닫기
- 우클릭으로 빠른 메뉴 접근

## 통계 확인

- **일별 그래프** - 시간대별 터치 패턴 분석
- **주별 그래프** - 요일별 터치 현황
- **캘린더** - 월별 터치 횟수 시각화
- **스트릭** - 연속 터치-프리 일수 확인

## 자주 묻는 질문

### 웹캠이 작동하지 않아요
- 다른 앱에서 웹캠을 사용 중인지 확인하세요
- 웹캠 드라이버가 최신인지 확인하세요
- 앱에 카메라 권한이 있는지 확인하세요

### 감지가 너무 민감해요 / 둔해요
- 설정에서 민감도를 조절하세요
- 트리거 시간을 늘리거나 줄여보세요

### 알림 소리를 바꾸고 싶어요
- 설정에서 알림 소리를 선택하거나 음소거할 수 있습니다
