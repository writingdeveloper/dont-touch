## 简介

Don't Touch 是一款帮助矫正无意识触摸或拔头发习惯（拔毛症/拔毛癖）的桌面应用程序。

通过网络摄像头实时追踪手部动作，当手靠近脸部或头部时发出警报，帮助您意识到这个习惯。

## 主要功能

- 🎥 **实时检测** - 通过摄像头实时追踪手部和面部位置
- 🔔 **即时警报** - 手靠近头部时发出声音和视觉警告
- 📊 **统计与追踪** - 记录和分析每日/每周触摸次数
- 📅 **日历视图** - 一目了然地查看月度习惯概况
- 🔥 **连续天数** - 追踪连续无触摸天数以保持动力
- ⚙️ **自定义设置** - 调整灵敏度、警报声音、冷却时间等
- 🌐 **多语言支持** - 韩语、英语、日语、中文、西班牙语、俄语
- 🖥️ **系统托盘** - 在后台静默运行
- 🔄 **自动更新检查** - 新版本发布时通知
- 📷 **预览切换** - 关闭摄像头预览以节省CPU（检测继续运行）

## 下载与安装

### Windows

1. 从 [Releases](https://github.com/writingdeveloper/dont-touch/releases) 页面下载 `DontTouch_Setup_x.x.x.exe`
2. 运行安装程序并按照安装向导操作
3. 从开始菜单或桌面快捷方式启动

**安装程序功能：**
- 桌面和开始菜单快捷方式
- Windows 启动时自动运行选项
- 通过控制面板轻松卸载
- 自动更新通知

> ⚠️ **Windows 安全警告**
>
> 由于此应用尚未进行代码签名，Windows 可能会显示安全警告：
>
> **Windows SmartScreen:**
> 1. 点击"更多信息"

>

> - 临时禁用：设置 → 隐私和安全性 → Windows 安全中心 → 应用和浏览器控制 → Smart App Control → 关闭

## 使用方法

### 基本使用

1. **启动应用** - 从开始菜单或桌面快捷方式启动
2. **开始监控** - 点击"开始"按钮
3. **正常工作** - 应用在后台监控
4. **接收警报** - 当手靠近头部时收到通知

### 调整设置

- **灵敏度** - 数值越低，检测越灵敏
- **触发时间** - 警报触发前的等待时间
- **冷却时间** - 防止连续警报的等待时间

### 系统托盘

- 关闭窗口后仍在托盘中运行
- 点击托盘图标打开/关闭窗口
- 右键点击访问快捷菜单

## 查看统计

- **日统计图** - 分析各时段的触摸模式
- **周统计图** - 查看每周各天的触摸情况
- **日历** - 可视化月度触摸次数
- **连续天数** - 查看连续无触摸天数

## 常见问题

### 摄像头无法工作
- 检查是否有其他应用正在使用摄像头
- 确保摄像头驱动程序是最新的
- 验证应用是否有摄像头权限

### 检测太灵敏/不够灵敏
- 在设置中调整灵敏度
- 尝试增加或减少触发时间

### 我想更改警报声音
- 您可以在设置中选择警报声音或静音

## 隐私保护

- 所有数据**仅存储在本地**
- 摄像头画面**不会发送到服务器**
- **无需网络连接**（离线运行）
